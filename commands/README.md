# commands/

Prompt templates with `$ARGUMENTS` placeholders, intended to be executed as
slash commands (Claude Code) or `akm command <ref>` invocations. Each
command is one prompt — short, parameterized, single-task.

For multi-step playbooks see [`workflows/`](../workflows). For interactive,
progressive-disclosure guides see [`skills/`](../skills).

Browse the full set with:

```bash
akm search --type command
```
